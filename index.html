Colour Compare Tool

Use the sections below to help to decide which colours to use when building applications.

  1. Scroll down the lists, each containing 140 colours.
  2. Click on the colours to display them in the panels above.
  3. Compare the left-hand and right-hand sections.
  4. Click the reset buttons to go back to the start.
